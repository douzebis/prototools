# prototools

A collection of protobuf utilities.

## Tools

### `prototext`

Lossless, bidirectional converter between binary protobuf wire format and
human-readable text.

→ See [`prototext/README.md`](prototext/README.md)

### `reproto`

Reconstructs `.proto` source files from compiled protobuf descriptor sets.

→ See [`reproto/README.md`](reproto/README.md)

### `protoscan`

Scans binary files for embedded protobuf `FileDescriptorProto` blobs.
